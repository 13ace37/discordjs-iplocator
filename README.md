# discordjs-iplocator
Geolocate IP / Resolve Domain

# Commands
- Geolocate IP - `.locate [IP]`
- Resolve Domain - `.resolve [DOMAIN]`

- `npm i`
- enter ur bot-token in the `config.json`
- `node iplocator.js`
